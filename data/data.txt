8) Why does this code not work?

}

Because the method 'getElementsByClassName' returns a node list, which you must iterate over in order use its contents.

Question:
9) What is the time complexity of this function?

const findIndex = (items, match) => {
  for (let i = 0, total = items.length; i < total; i++)
    if (items[i] == match)
      return i;
   return -1;
};

Answer: O(N)
